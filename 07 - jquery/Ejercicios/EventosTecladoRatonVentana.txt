Eventos de jQuery: Teclado
Ejercicios:
1-Detecta cuando se presiona la tecla "Enter" en un <input> y muestra un mensaje en la consola.
2-Muestra un mensaje cada vez que el usuario escriba en un campo de texto con id #inputTexto.
3-Imprime en la consola el código de la tecla presionada en un campo de texto.
4-Detecta cuando el usuario presiona la tecla "Esc" y limpia el contenido de un campo con id #campo.
5-Cambia el fondo de un campo de texto a amarillo cuando se presiona la tecla "Tab".
6-Añadir un evento keydown a un campo de entrada que muestre el carácter que se ha escrito.
7-Detecta cuando se escribe en un campo con id #email y verifica si tiene un "@" y muestra un mensaje.
8-Detecta cuando se presiona una tecla en un campo de texto con id #busqueda y muestra un mensaje en consola.
9-Muestra el número de caracteres restantes cuando el usuario escribe en un campo con un límite de 10 caracteres.
10-Detecta cuando el usuario presiona la tecla "Shift" y cambia el color de fondo de un <p>.



Eventos de jQuery: Ratón

Ejercicios:
11-Detecta cuando el usuario pasa el ratón sobre un <div> con id #area y cambia su fondo a verde.
12-Cuando el usuario hace clic en un <button>, cambia el texto a "Clicado".
13-Detecta el movimiento del ratón sobre un área con id #contenedor y muestra las coordenadas del ratón en consola.
14-Haz que un <p> con clase .item se haga más grande al pasar el ratón por encima.
15-Haz que un <div> cambie de color cuando el ratón entre y vuelva al color original cuando el ratón salga.
16-Detecta el clic derecho del ratón sobre un <span> y muestra un mensaje.
17-Haz que una imagen con id #foto se mueva con el ratón dentro de un contenedor con id #area.
18-Detecta cuando el usuario hace doble clic sobre un <p> y cambia su texto.
19-Haz que el contenido de un <div> se desplace hacia arriba cuando el ratón entre en el área.
20-Detecta cuando el ratón sale de un área con id #zona y muestra un mensaje de alerta.



Eventos de jQuery: Ventana

Ejercicios:
21-Detecta cuando la ventana se redimensiona y muestra un mensaje en la consola.
22-Detecta cuando la ventana se carga y cambia el fondo de la página a azul.
23-Detecta cuando se minimiza la ventana y muestra un mensaje en la consola.
24-Haz que el contenido de un <div> se adapte al tamaño de la ventana.
25-Muestra el tamaño de la ventana cada vez que se redimensiona.
26-Detecta cuando la ventana está a punto de cerrarse y muestra un mensaje de confirmación.
27-Haz que un elemento con id #contenedor se muestre u oculte cuando se haga un scroll en la página.
28-Detecta cuando la ventana se desplaza y muestra la posición en la consola.
29-Haz que un <div> cambie de tamaño cuando se redimensiona la ventana.
30-Detecta si la ventana es redimensionada a un tamaño específico y muestra un mensaje.
